Bloom Filter

Quick Start

# Clone and build
git clone https://github.com/sauravbhattacharya001/Ocaml-sample-code.git
cd Ocaml-sample-code
make run

# Or use Docker
docker build -t ocaml-samples .
docker run --rm ocaml-samples
